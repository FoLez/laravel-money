# `getCurrency()`
A shortcut for getting the currency.

## Methods

### `getCurrency()`

**Returns**: `Currency`

## Usage

The full way:
```php
$money = money(1000);
$money->settings()->getCurrency();
```

The shortcut:
```php
$money = money(1000);
$money->getCurrency(); // PostScripton\Money\Currency object
```
